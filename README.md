SISTEMA DE GESTIÓN DE VENTAS 

Desarrollado en C# (.NET) utilizando SQL Server como base de datos.
El sistema permite administrar productos, clientes y ventas mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar), 
facilitando el control del inventario y el registro de transacciones.

TECNOLOGÍAS UTILIZADAS:

C#.

SQL Server.

Windows Forms.

Visual Studio.

FUNCIONALIDADES PRINCIPALES

Crear nuevos registros de productos.

Consultar información almacenada.

Actualizar datos existentes.

Eliminar registros del sistema.

Registrar ventas.
